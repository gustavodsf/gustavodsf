## 🏆 What I've Built

- 🎙️ **Voice-ordering AI system** — integrated an LLM with traditional POS platforms via a Spring Boot backend-for-frontend (real-time audio streams, session state, voice-event orchestration) plus an Angular WebSocket state machine and a Redis-backed menu cache for sub-second cart pricing.
- 🔐 **Crypto risk & security** — transaction-pattern risk rules and threat flagging for a global exchange (Spring Boot, MySQL, microservices) in a zero-tolerance domain.
- 💳 **Payments reliability** — hardened a Node.js / Express / GraphQL BFF: cut retry volume **7%** and memory **4%**, and added integration component tests.
- 🌳 **Geospatial platform at scale** — migrated a monolith to Nest.js microservices, scaling processing from 1,000 to 100,000 locations (**100x throughput**) with AWS SQS async processing.
- 🏦 **Banking-as-a-service observability** — custom Spring Boot Starter for unified logging (Splunk + Kafka), distributed tracing (Sleuth/Zipkin), and Resilience4J circuit breakers across a microservices fleet.
- 🧠 **ML pipelines** — music-detection model for media metadata (Librosa, scikit-learn) and predictive/optimization models for industrial analytics (gradient boosting, neural nets, clustering/RL).
